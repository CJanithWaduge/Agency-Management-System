# Portfolio assets

Place screenshots for the Upwork portfolio in this folder. Recommended filenames and specs:
- `dashboard.png` — 1200x800, shows main dashboard with charts
- `sales.png` — 1200x800, shows sales list / invoice view
- `inventory.png` — 1200x800, shows inventory grid and item details
- `reports.png` — 1200x800, shows report preview or export
- `settings.png` — 1200x800, shows authentication/settings UI

Tips:
- Use a clean crop, no personal data, and blur any sensitive fields.
- Export at 72–150 DPI for web, keep filesize reasonable (<300 KB each) while preserving quality.

How to capture:
- Run the app locally (`npm run dev`) or open the Electron build, take screenshots, then place them here.
