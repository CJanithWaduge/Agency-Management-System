# Agency Management System v1

Short blurb (150 chars):

Full description (copy for Upwork portfolio):
Agency Management System v1 is a complete desktop/web application designed to streamline agency operations. Built with React and Vite for the frontend and packaged with Electron for a cross-platform desktop experience, the app includes modules for Sales, Inventory, Expenses, Creditors, Analytics, and Reporting. It features user authentication, interactive charts, CSV/print-ready reports, and a lightweight Node-based backend for local data persistence.

Problem solved:
- Agencies often juggle multiple spreadsheets and disconnected tools. This app consolidates core workflows (sales, inventory, expenses, reporting) into a single, offline-capable desktop/web interface.

My role:
- Full-stack development: UI, client-side logic, Electron packaging, and simple Node-based local persistence.

Highlights / features:
- Dashboard with aggregated KPIs and interactive charts
- Sales, Inventory, Expenses modules with filtering and search
- History and statement export (CSV/print)
- Authentication and user settings
- Responsive UI built with Tailwind + custom components

Tech stack:
- Frontend: React, Vite, Tailwind CSS
- Desktop: Electron
- Backend: Node (local/embedded), simple file-based DB (see `src/backend/database.cjs`)
- Language: JavaScript

How to run (quick):
- Development (web): `npm run dev`
- Desktop build / run: `npm run electron` (project contains an Electron wrapper)

Key results:
- Replaced fragmented spreadsheets with one app covering key business flows
- Improved visibility with analytics and historical reporting
