- Full description (paste this into the "Project description" box):
Agency Management System v1 is a complete desktop/web application designed to streamline agency operations. Built with React and Vite for the frontend and packaged with Electron for a cross-platform desktop experience, the app includes modules for Sales, Inventory, Expenses, Creditors, Analytics, and Reporting. It features user authentication, interactive charts, CSV/print-ready reports, and a lightweight Node-based backend for local data persistence. I handled full-stack development including UI, client-side logic, packaging, and integrating reporting/export functionality.
